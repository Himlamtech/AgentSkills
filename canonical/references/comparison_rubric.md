# Comparison Rubric

Score each option on:
- capability fit
- quality
- latency
- cost
- maturity
- integration effort
- security
- observability

Use one line per option:
- `Adopt` if it clearly wins and fits current architecture.
- `Pilot` if promising but needs validation.
- `Watch` if useful but not ready.
- `Reject` if cost, risk, or mismatch is too high.
